# fps-prototype
Roblox用FPSプロトタイプ（1v1武器バランス検証）
# FPS プロトタイプ仕様（まとめ）

概要:
- プラットフォーム: Roblox（プロトタイプ）
- 目的: 1v1 で武器バランスを検証する最小MVPを作成する
- 主要武器: AR / SMG / SG
- プレイヤー体力: 200 HP

構成:
- SPEC/ : 仕様の詳細（MVP, 武器パラメータ, ロードマップ）
- ASSETS/ : 3Dモデルやチェックリスト
- TEST/ : プレイテスト計画とログフォーマット

次のアクション:
1. プロトタイプ用アセットを調達（Roblox Asset Library か自作）
2. MVP を実装して内部テスト（1v1、30セッション目安）
3. KPI に基づきチューニング
